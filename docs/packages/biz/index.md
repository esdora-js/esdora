# 业务工具 (@esdora/biz)

欢迎来到 `@esdora/biz` 的世界！这是 `Dora Pocket` 中专注于业务场景的工具库。

本模块封装了业务开发中常用的工具函数，旨在简化业务逻辑的实现，提升开发效率。

## 核心理念

我们遵循**实用至上**的原则。专注于提供那些在业务开发中频繁使用、能够解决实际问题的工具函数，让你的业务代码更加简洁优雅。

## 功能模块

### 查询工具 (Query)

提供完整的 URL 查询字符串处理能力，包括：

- **解析工具**：将查询字符串转换为对象
- **序列化工具**：将对象转换为查询字符串
- **浏览器集成**：直接从 `window.location.search` 解析查询参数
- **参数合并**：智能合并多个查询参数对象

## 浏览函数

你可以通过左侧的侧边栏，按类别浏览所有可用的函数。
