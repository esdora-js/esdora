# Project Overview

## Identity

- Name: esdora (Dora Pocket / 四次元口袋)
- Type: TypeScript utility library monorepo
- Philosophy: Practical-first, zero-dependency core, ESM + CJS dual output

## Quick Reference

| Task | Command |
|------|---------|
| Install deps | `pnpm install` |
| Build all | `pnpm build` |
| Test all | `pnpm test` |
| Lint | `pnpm lint` |
| Typecheck | `pnpm typecheck` |
| Dev mode | `pnpm dev` |
| Docs site | `pnpm docs` |

## Documentation Layers

- Human API docs: `docs/` (VitePress site)
- AI dev guide: `.claude/rules/` (this directory)
